Processing RPG Game

Mystery Girl


Mystery Girl is a fun little project that I worked on long time ago. I wanted to create an RPG-maker looking game with processing as my developing platform. It is a horror game that has 2 different endings that the player is able to discover.

-CLIENT

Self Project


-DATE

2013


-TECHNOLOGY

Processing

-MEMBERS

Eldwin

My process


I started the project by ideating what I wanted to create. Back in the time, I always admired the people who made interesting story game with RPG maker, therefore, I decided to create a horror game using processing. First, I created a general story that I wanted to tell the audience; writing down what I wanted for each level, what type of encounter that the player will experience, what are the endings, etc.


After that, I started creating multiple assets using processing, with simple geometric shapes (this includes character and everything within the environment). Here is where I met my biggest obstacle, programming. This project was my first time ever using processing platform, I had a lot of trouble programming the interaction between the player and objects. However, with the help of peers from the internet, I was able to successfully move forward.


Overall, I consider this as a unique project to me, since this is when I first started visualizing and creating my own game. It was a very fun and exhausting experience since I had a time schedule of only 2 weeks to complete it. I learned a lot about programming and game design during that short period of time through both research and peer suggestion.
